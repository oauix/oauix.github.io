能用这个处理输出输入流就ok了

[输入流应用:]

	FileInputStream fis= new FileInputStream("osw.txt");
		
	InputStreamReader isr= new InputStreamReader(fis,"UTF-8");//转换流
	
	BufferedReader br= new BufferedReader(isr);
		
	int d = -1;
	while((d = isr.read())!=-1){
		System.out.print((char)d);
	}

[输出流应用:]

	FileOutputStream fos= new FileOutputStream("osw.txt",true);//true不覆盖，默认false覆盖
		
	OutputStreamWriter osw= new OutputStreamWriter(fos,"UTF-8");//转换流,高级流

	PrintWriter pw = new PrintWriter(osw,true);//true自动刷新输出,默认是缓存的,可以用wp.flush()强制输出
		
	pw.println("我爱北京天安门,");

	pw.println("吹啊吹啊我的骄傲放纵!");
