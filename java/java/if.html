分支结构:

1)if:1条路 if(flag){}

2)if...else:2条路 if(flag){}else{}

3)if...else if:多条路if(flag){}else if{}else if{} 是if()else()中套用

4)switch...case:多条路： 如果需求正好是对整数来判断相等，首选switch...case

格式：switch(){case a:break;case b:break;default:break;} 中间的abc不能为变量，要是常量

优点:效率高、结构清晰

缺点:条件要是整数、并相等

break:跳出switch

任何复杂的程序逻辑都可以通过三种结构来实现:

1)顺序结构:从上往下逐行执行，每句必走

2)分支结构:有条件的执行某语句，并非每句必走
