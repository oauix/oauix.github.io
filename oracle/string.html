操作字符串

	  1.concar 与 ‘||’
	  连接两个字符串，用法：concat(char1,char2),char1 || char2

	  2.length(char); 获取字符串长度

	  3.upper(char) lower(char) initcap(char) 转大写、小写、首字母大写 initcap里面是多个单词就都首字母大写

	  4.taim(c2 from c1) ltrim(c1[,c2]) rtrim(c1[,c2]) 从c1中前后截取c2[字符不能是字符串]、在c1字符串left[最前面]截取c2，在c1字符串right[z最后面]截取c2；

	  5.lpad(char1,n,char2) rpad(char1,n,char2) 分别从char1左右端用char2补足n位 又有对齐效果，
	  案例：select lpad(ename,10,' ') as "bb" from emp;这就右对齐效果了

	  6.substr(char,m[,n]) 截取一个char字符串，这个是从m位置开始，m可以为负数，意思倒数几位，n不是位置结束，这里是长度，数据库字符串下标是从1开始的。

		
	  7.instr （char1,char[,n[,m]]）从char1中查char2的位置，n是从什么位置开始查找，当n为负数时，m是逆时针第几次，m是指字符出现的次数。
		
	  8.round(n[,m]) 用于四舍五入-select round(45.678,-1) from dual;//50,m是看的位数
	  9.trunc(n[,m]) 跟round一样，但不是四舍五入，后面不什么都舍去

	 10.mod(m,n) -返回m除以n的余数，n为0时，就直接返回m

	 11.ceil(n) 取大于或等于n的最小整数 select ceit(45.678) from dual //46
	 12.floot(n)取小于或等于n的最大整数 select floor(45.678) from dual //45

	      1982年01月01号显示。格式都是DD-MON-RR(默认格式)
