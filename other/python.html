画笔控制函数：

Turtle.forward(d) 别名turtle.fd(d) 向前行进，走直接

数据处理函数

Pow(x,y) 计算x的y次方

Round(x,d) 对x四舍五入，d为小数截取位数

<a>e<b> 解释：ax10的b次方

复数 x的2次方=-1，由于j=根号-1，数学里这个数是不存在的，python里把它作为复数

x%y 取余 X//y x除y取整数 x**y x的y次方，如果y是小数 就是x的y根号

Abs（x） 得到x绝对值

Divmod(x,y) 同时得到商和余

Pow（x,y,z） 等同于x**y%z

Max（x1,x2,x3...）比较一组数的大小，然后返回最大的

Min（x1,x2,x3...）比较一组数的大小，然后返回最小的

数据类型转换：Int（x） float（x） complex(x)//x变为复数

Python第三方打包库——PyInstaller：

pip install pyinstaller #安装第三方库，安装之后会提示successfully installed

在PyInstaller安装完成后，会在Python目录下额Scripts目录下生成运行文件。此时在命令行下使用Pyinstaller -F <.py文件地址>命令就可以打包程序

如：PyInstaller -F D:/xyu/12.py

打包完成后，除了用于存放.exe文件的dist文件夹，另外两个存放临时文件的文件夹均可手动删除。
