数组

相同数据类型元素的集合称之为数组,是一种数据类型(引用类型)


数组的初始化:

int[] arr = new int[4]; //0,0,0,0

int[] arr = {1,4,6,8}; //1,4,6,8不能分开写

int[] arr = new int[]{1,4,6,8}; //1,4,6,8

int[][] arr = new int[][];int类型的不规则数组,n行n列

int[][] arr = new int[2][3];int类型二维数组2行3列的空间大小

arr = new int[][]{{1,2,3},{1,2,3}};

int[][] arr = new int[4][];int类型不规则数组,4行n列

迭代器[Iterater]实现遍历：

while(数组或集合变量.hasNext()){

System.out.println(数组或集合变量.next());

}

a = Arrays.copyOf(a,a.length+1); //常用于扩容
