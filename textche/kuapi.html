String ：字符串是不变对象,即:字符串对象创建后内容不可变，改变内容会创建新对象，再创建时会先查询常量池，无才创建


正则表达式： [0123456789]、[0-9]、\d:代表一个0-9的一个字符；[a-z]:一个a-z的一个字符；[^12]:除了12其他的一个字符 ^取非；Boolean a=object.matches(str);object为正则表达式；[0-9][0-9][0-9]--简写--[0-9]{3};//{3}是数量值；

正则表达式之数量词：

x？ 0个或一个；x* 0个或多个；x+ 1个或多个；x{n} n个字符；x{n,} n个或多个；x{n,m} n到m个

. 任意一个字符

\d 任意一个数字字符，相当于[0-9]

\w 任意一个字母或数字，相当于[a-zA-Z0-9_]

\s 任意空符，相当于[\t\n\xoB\f\r]

\D

\W 与上意思相反

\S

“^”与”$”边界匹配符:

^:开始

$:结束

相对而言的：用查询功能：ctrl+f，String str=”abcd_123456789”;边界匹配当他们等时，才返回true


String常用方法:

1.length() 返回字符串长度

2.indexOf()--取字符位置

* int indexOf(String str) 重载：int indexOf(String str,a) a为位置。

* 返回给定字符串在当前字符串中的位置,若当前字符串不含有给定字符串则返回-1.

* int a=object.indexOf(str);

3.substring()--截取字符

* String substring(int start,int end)

* 截取当前字符串指定范围内的字符串并返回.

* 在java api中通常使用两个数字表示范围时,都是"含头不含尾"的.

* String a=object.subString(2,4);

4.charAt()--获取字符

* char charAt(int index)

* 返回当前字符串中指定位置的字符

* char a=object.charAt(5);

5.trim()--去前后空白符

* String trim()

* 去除当前字符串两边的空白字符

* String a=object.trim();

6.statsWith() endsWith(String str)--前后匹配

* boolean startsWith(String str)

* boolean endsWith(String str)

* 判断字符串是否是以给定字符串开始或结尾的

* boolean a=object.startsWith(“ab”); boolean b=object.endWith(“ab”);

7.toUpperCase()toLowerCase()--转换大小写[限字母]

* String toUpperCase()

* String toLowerCase()

* 将当前字符串中的英文部分转换为全大写或全小写

* String a=object. toUpperCase()

8.valueOf()--转换为字符串

* static String valueOf(...)

* String提供了一组重载的静态方法valueOf

* 作用是将java其他类型转换为字符串

* 常用于将基本类型转换为字符串

* String a=String.valueOf(field);

9.toCharArray()--换行成数字

* Char[] str=s.toCharArray();//把s字符串转换为char数组

10.equals("str")--字符串对比

* 判断string内容是否相同：用:字符串.equals(对比字符串);

[String:正则API方法]

11.matches()//匹配方法

* boolean flag=Object.matches("str");

12.split(a)//拆分方法，a是正则表达式。

* String[] array=object.split(正则表达式);拆分后边几个数组了

13.replaceAll();//替换方法

* String a=object.replaceAll(“正则表达式”,”要替换的内容”);

[String不可改变,因此java没有向字符串中间添加/删除的方法]

StringBuilder：对于String不能更改，推出该类，【 java.lang.StringBuilder】

* StringBuilder是为了解决频繁修改字符串对系统资源开销问题.

* StringBuilder内部是一个可变的字符数组,可以通过扩容来实现长度变换,而不是每次创建新对象


StringBuilder提供了方便修改字符串内容的方法.

toString()

toString()返回对象的东西，将其转换为字符串输出，但引用类型的东西是抽象的，所以最好指定一下需要返回的东西

该方法是sun公司在object定义了,是返回：类名@散列吗;,大多数java内部类已重写，自己定义类最好要自己重写。最好是返回重要的数据


equals()

跟toString()一样，不能比较引用类型的东西，如果一定要比较，他们是比较引用上的指向的内存地址，是不是同一个指向

Date

* Date库类(从java1.1已不用了)

* getTime()与setTime();前者将date类型值转换为long，后者相反

	import java.util.Date;
	/**
	*使用Date输出当前系统时间，并告知3天后这一刻的时间
	*
	*/

   public class test{
		public static void main(String[] agrs){
			Date date=new Date();//获取一个时间节点，当前系统时间
			System.out.println(date);
			long time = date.getTime();//转换为自1970.1.1到截取时间的毫秒数
			System.out.println(time);
			time+=24*60*60*1000*3;
			date.setTime(time);//再转换为日期格式
			System.out.println(date);
		}
   }

SimpleDateFormat[格式类，java.text.SimpleDateFormat]

自定义格式：object.format();[日期格式>自定义]object.parse();[相反]

	import java.util.Date;
	import java.text.SimpleDateFormat;
	/**
	*将当前系统时间转换以"yyyy-MM-dd HH:mm:ss"格式输出
	*
	*/
	public class tests{
		public static void main(String[] agrs){
			Date date = new Date();
			SimpleDateFormat sd = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
			String str = sd.format(date);//将日期格式转换为指定格式
			Date sb= sd.parse(str);//再将指定格式反转换为日期格式
			System.out.println(str+","+sb);
		}
	}
