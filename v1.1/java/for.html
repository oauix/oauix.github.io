循环结构:

1)while:先判断后执行，有可能一次都不执行 --当

2)do...while:先执行后判断，至少会执行一次 ---直到什么时候(或情况)

要素1与要素3相同时，首选do...while

3)for:应用率最高，固定次数循环

循环三要素:

1)循环变量的初始化 //循环变量：在循环过程中反复改变的那个量

2)循环的条件(以循环变量为基础)

3)循环变量的改变(向着循环的结束变)

循环变量:在循环过程中反复改变的那个量


增强for-迭代器[扩展]

迭代器[Iterater]实现遍历：

while(数组或集合变量.hasNext()){

System.out.println(数组或集合变量.next());

}

增强for[又称新循环]

for(数据类型 变量 : 数组[集合]){//数据类型是数组或集合中数据类型

System.out.println(变量);//循环一次，获取一次值，这个存储在这个变量中

}

数组没有迭代器 但可以增强for循环,是因为java有一个强大的jmv虚拟机,在编译时就将增强for循环转换为普通的for循环
