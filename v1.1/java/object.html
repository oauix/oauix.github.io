Object

* 没有指定继承的类，默认都继承Object

* 重写toString()与equals()方法

toString()

toString()返回对象的东西，将其转换为字符串输出，但引用类型的东西是抽象的，所以最好指定一下需要返回的东西

该方法是sun公司在object定义了,是返回：类名@散列吗;,大多数java内部类已重写，自己定义类最好要自己重写。最好是返回重要的数据

equals()

跟toString()一样，不能比较引用类型的东西，如果一定要比较，他们是比较引用上的指向的内存地址，是不是同一个指向

重写格式：

boolean equals(Object obj){

if(obj==null){

return false;

}

if(this==obj){

return true;

}

if(obj instanceof 要对比的类Oo){

Oo oo=(String) obj;

return oo.name==obj.name;

}else{

return false;

}

}
