第二课，html常用标签(重点难点)

注意(新加)部分：

1.始终为属性值加引号

2.当显示页面时，浏览器会移除源代码中多余的空格和空行。所有连续的空格或空行都会被算作一个空格。需要注意的是，HTML 代码中的所有连续的空行（换行）也被显示为一个空格。

3.<big>变淡标签、<pre>预格式文本,空格会全显示，很适合显示计算机代码。

<html>标签

html标签是双标签，以<html>开始点，以</html>结束点，在它们之间是文档的头部和主体.

<!doctype html>

声明不是 HTML 标签；它是指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令。自己理解：适合哪种浏览器，一种兼容浏览器声明

<head>

自己理解：双标签，html的头部，通告浏览器一些信息，不显示；<head>>标签用于定义文档的头部，它是所有头部元素的容器。<head>中的元素可以引用脚、指示浏览器在哪里找到样式表、提供元信息等等。文档的头部描述了文档的各种属性和信息，包括文档的标题、在 Web 中的位置以及和其他文档的关系等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者。

<meta>

单标签，<meta>>元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词。<meta>标签位于文档的头部，不包含任何内容。<meta>标签的属性定义了与文档相关联的名称/值对。

<title>

双标签，浏览器显示出来的标题

<body>

自己理解：双标签，身体，浏览器显示部分；body 元素定义文档的主体。 body 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等）。

<p>段落标签

双标签，顾名思义用于段落，他会在正文前后加一行空行

特殊符号

小于号[<]代码：-----&lt;------

大于号[>]代码：-----&gt;------

双引号["]代码：-----&quot;-----

版权号[©]代码：----&copy;-----

空格号[ ]代码：-----&nbsp;-----

<font>标签

<font>标签常用格式：

<font size="字体大小" color="字体颜色" face="字体类型">文字</font>
注：但是<font>里的属性不推荐用，一般用css来实现

<b>、<strong>加粗、<i>斜体、<span>、<u>下划线

<b><strong>双标签，加粗，<strong>效果更加好但不常用
<i><span>单标签，<span>跟css很实用，不加样式没有效果

<br/>换行标签、<hr/>分割线标签

<br/>单标签，实现换一行显示
<hr/>单标签，一条分割线，可以用样式调整他显示方式，很实用

<a>超级链接标签

超级链接常用格式：

<a href="URL" target="打开方式">文字</a>

URL-绝对路径、#id(描点链接,跟<span id=id><span>配用)、+文件名(即跳转该文件同目录下文件)、../文件名(即跳转该文件上一目录下文件)

打开方式：_blank(在新窗口中打开)、_parent(在父框架集中打开)、_self(在相同的框架中打开 默认)、_top(在整个窗口中打开)、framename(在指定的框架中打开)

<marquee>滚动标签

<marquee>常用格式：

<marquee direction="方向" scrollamount=“速度”>内容<marquee>
注：这个标签属性很多，常用的有方向、速度、鼠标放上离开。
